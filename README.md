# SplitPay Android App

SplitPay is an Android application designed to track shared expenses and actually send payments between friends and groups.

Features:
- Track incoming and outgoing payments
- Split expenses between multiple people
- Filter transactions (All, Incoming, Outgoing, Auto-Pay)
- View balance summary (owed vs owe)

Built using:
- Java
- Android Studio
- RecyclerView
- Material UI components
